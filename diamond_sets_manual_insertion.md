## Admin Page Insertion Steps

### Step 1: Navigate to Admin Dashboard
1. Log into your admin account
2. Go to Products section
3. Click "Add New Product"

### Step 2: Fill Product Information
1. **Product Name:** Use the exact names listed above
2. **Description:** Copy the descriptions provided
3. **Base Price:** Set to ₦70,000 (All sizes use same base price)
4. **Category:** Select "diamond-set" or create if not exists
5. **Gender:** Select "unisex"
6. **SKU Prefix:** Use the specified prefixes (DS-NB or DS-AP)
7. **Mark as New Release:** Check this option

### Step 3: Add Color Variants
1. Click "Add Color Variant"
2. For each color variant:
   - Select the color from your color list
   - Upload corresponding product images
   - Set stock quantities for each size

### Step 4: Set Stock Levels
For each color variant, set the stock quantities:
- Small (S): As specified above
- Medium (M): As specified above
- Large (L): As specified above
- Extra Large (XL): As specified above

### Step 5: Upload Images
- Upload high-quality images for each color variant
- Ensure images show both colors clearly
- Use the IMG_4566 series for Navy Blue & Sky Blue
- Use the IMG_4571 series for Ash & Pink

### Step 6: Save and Publish
1. Review all details
2. Click "Save Product"
3. Make sure product is set to "Active"
